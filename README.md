# console

- apps:

  - [@zws/back](apps/front/README.md) nestjs

# installation

```bash
# build & run your local environment
$ docker-compose up

# install packages
$ pnpm install
```

# develop

```bash
# build the necessary packages
pnpm --filter utils build

# run all in dev mode
$ pnpm start
```
