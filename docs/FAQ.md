# FAQ

## Who is this project for?
This project is intended for developers experimenting with fully homomorphic encryption on EVM-compatible chains.

## Do I need cryptography experience?
No. Basic smart contract knowledge is enough to get started.

## Is this production-ready?
The project is under active development and should be used with caution in production environments.

## Where can I ask questions?
Please use GitHub Issues for questions and discussions.
