# Support

This document explains how to get help with fhevm.

## Where to ask questions
- GitHub Issues: for confirmed bugs or feature requests
- Discussions (if enabled): for general questions and ideas

## Before asking for help
Please prepare:
- fhevm version
- Network used
- Minimal reproduction steps
- Relevant logs (without secrets)

## What not to include
- Private keys or credentials
- Sensitive infrastructure details
