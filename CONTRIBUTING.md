# Contributing Guide

Thank you for your interest in contributing to FHEVM!

## How to contribute

- Check existing issues: https://github.com/zama-ai/fhevm/issues  
- Open a new issue if you want to report a bug or propose a feature  
- Fork the repository and create a dedicated branch for your contribution  
- Ensure your changes are clear, documented, and tested when relevant

## Pull requests

- Keep PRs focused and easy to review  
- Provide a short description of what the change improves  
- Link to the related issue if applicable  

## Communication

For general questions, use GitHub Discussions or the official Zama community channels.

We appreciate your contributions and support for the FHEVM ecosystem!
