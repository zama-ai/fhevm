# Contributing to FHEVM

Thank you for your interest in contributing to FHEVM, the core framework of the Zama Confidential Blockchain Protocol.

## Ways to contribute

- Open GitHub issues to report bugs, suggest improvements, or flag documentation problems.  
- Request to become an approved contributor by emailing `hello@zama.ai`.  

Issues are the best place to start discussions and share feedback with the maintainers.

## Pull requests

Only approved contributors who have signed the Contributor License Agreement (CLA) can open pull requests.

If you would like to work on code changes:

1. Contact the team at `hello@zama.ai` and mention your intended contribution.  
2. Follow the CLA process described by the maintainers.  
3. Keep PRs focused, with clear descriptions and links to related issues where possible.
