# Security disclosure for FHEVM

Responsible disclosure is important to protect users of the FHEVM stack.

If you discover a vulnerability, please do not open a public GitHub issue. Instead:

1. Send an email to security@zama.ai with a detailed description of the issue.
2. Include steps to reproduce, the impact, and potential mitigations.
3. We will acknowledge your report within two working days and work to fix the problem promptly.

We appreciate your help in keeping FHEVM safe.
