# Solana PoC Development Issues

The rolling issue ledger that used to live here has been distilled into durable docs:

- `docs/DEVELOPMENT_HISTORY.md` — the hardening catalog (what the ~176 resolved issues bought,
  grouped by theme) and the status snapshot.
- `docs/EVM_PARITY.md` — the EVM->Solana parity map and the open / product-open items.
- `docs/DESIGN_DECISIONS.md` — the stable design rationale (DD-001..DD-013).

The granular enforcement those issues added lives in the program code and its rejection tests
(see `docs/TESTING.md`).
